# User Profile App

A React application that manages a user profile with nested state using the spread operator to ensure immutability.

## Features

- Displays a user profile with name, email, and nested address fields
- Input fields to update street, city, and country
- Updates nested state immutably using the spread operator
- Real-time display of current profile information

## How to Run

1. Clone the repository:
   git clone <your-repo-url>

2. Navigate into the project folder:
   cd profile-app

3. Install dependencies:
   npm install

4. Start the development server:
   npm run dev

5. Open your browser and go to:
   http://localhost:5173

## Project Structure

- src/UserProfile.jsx — main component with nested state and updateAddress function
- src/App.jsx — root component that renders UserProfile

## Built With
