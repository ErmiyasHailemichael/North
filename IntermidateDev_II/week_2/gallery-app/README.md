# Recipe Gallery App

A React application that displays an interactive recipe image gallery built with Vite.

## Features

- Browse through 5 recipe images using Next and Previous buttons
- Boundary checks prevent navigation beyond the first or last image
- Displays image description and current position (e.g. 1/5)

## How to Run

1. Clone the repository:
   git clone <your-repo-url>

2. Navigate into the project folder:
   cd gallery-app

3. Install dependencies:
   npm install

4. Start the development server:
   npm run dev

5. Open your browser and go to:
   http://localhost:5173

## Project Structure

- src/Gallery.jsx — main gallery component with state and navigation logic
- src/App.jsx — root component that renders the Gallery

## Built With

- React
- Vite
