# Remix Blog App

A blog application built with React Router (formerly Remix) that includes a homepage listing blog posts, individual post pages, and nested category routing.

## Tech Stack
- React Router v7
- TypeScript

## Pages
- `/` - Homepage listing all blog posts
- `/blog/:slug` - Individual blog post page
- `/blog/category/:tag` - Filter posts by category

## How to Run

1. Install dependencies:
   npm install

2. Run the development server:
   npm run dev

3. Open your browser at:
   http://localhost:5173
