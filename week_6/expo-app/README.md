# Expo App

A cross-platform mobile app built with Expo and React Native that features a tab navigator with a Home tab displaying posts from a public API and an About tab.

## Tech Stack
- Expo
- React Native
- TypeScript

## Tabs
- `Home` - Displays a list of posts fetched from JSONPlaceholder API
- `About` - Shows bio, skills, and contact information

## How to Run

1. Install dependencies:
   npm install

2. Start the development server:
   npx expo start

3. Options to view the app:
   - Press `w` to open in web browser
   - Scan the QR code with Expo Go app on your phone
