# Gatsby Portfolio App

A portfolio website built with Gatsby that showcases projects and includes an about page. Uses Gatsby's GraphQL data layer and component-based architecture.

## Tech Stack
- Gatsby
- JavaScript
- GraphQL

## Pages
- `/` - Homepage with project cards
- `/about` - About page with bio, skills, and contact info

## How to Run

1. Install dependencies:
   npm install

2. Run the development server:
   npm run develop

3. Open your browser at:
   http://localhost:8000
