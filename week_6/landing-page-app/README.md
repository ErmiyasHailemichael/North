# Landing Page App

A simple landing page built with Next.js that includes a users list and dynamic user profiles fetched from a public API.

## Tech Stack
- Next.js
- TypeScript

## Pages
- `/` - Landing page with header, sections, and footer
- `/users` - List of users fetched from JSONPlaceholder API
- `/users/[id]` - Individual user profile page

## How to Run

1. Install dependencies:
   npm install

2. Run the development server:
   npm run dev

3. Open your browser at:
   http://localhost:3000
